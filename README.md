# Business Problem
- The need for Customer Segmentation with RFM Analysis with e-commerce site data.
# Data Story
- The dataset has information on 100,000 orders made from 2016 to 2018 across multiple marketplaces in Brazil. 
  Its features allow viewing an order from multiple dimensions: order number, customer number, total price paid, and date. The dataset consists of information derived from customers' past shopping behavior. 
  # Variables
- Order_id: Order number
- Customer_unique_id: Customer number
- Price: Total price paid by the customer
- Order_purchase_timestamp: Date of the customer purchase
